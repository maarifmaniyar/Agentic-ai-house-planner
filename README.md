# 🏠 Agentic AI House Planner

An intelligent **Agentic AI-powered House Planning System** built using **CrewAI, Ollama and Streamlit**.

This project simulates collaboration between multiple AI agents to generate:

- House layout planning
- Interior suggestions
- Cost estimation
- Top-view floor plan
- Downloadable PDF report

---

## 🚀 Live Demo
🔗 Streamlit App:(https://agentic-ai-house-planner-maarifmaniyar.streamlit.app/))

---

## 📌 Problem Statement
Traditional house planning requires consultations with multiple experts.

This project uses **Agentic AI** where specialized agents collaborate autonomously to assist in conceptual home planning.

---

## ✨ Features

- Multi-Agent Architecture using CrewAI
- Smart House Requirement Analysis
- Floor Plan Generation (Top View)
- Budget-Oriented Planning
- Professional Streamlit UI
- PDF House Plan Download
- Local LLM Support with Ollama

---

## 🧠 Agents Used

### 1. Requirement Analyzer Agent
Analyzes user requirements:
- Plot size
- Number of rooms
- Budget
- Design style

### 2. Architect Agent
Creates house layout recommendations.

### 3. Interior Design Agent
Suggests aesthetic and furniture ideas.

### 4. Cost Estimation Agent
Provides construction budget estimation.

### 5. Report Generator Agent
Compiles final structured house planning report.
